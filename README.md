# SimpleHorizLevel
Horizontal Level Chart 

## Usage

`<canvas id="shl1"></canvas>`




```javascript
function() {
    $('#shl1').simpleHorizLevel({ 
        data: [
            { name: first', value: 22, color: '#fc4c2d' },
            { name: second', value: 332, color: '#118f14'}
            ],
        options: {lineColor: 'black'}
        });
            
}
